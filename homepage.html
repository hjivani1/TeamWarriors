Soccer Game

Rules of the Game

  • 1. The goal of the game is to start from one and the end goal is to score.

  • 2. The user need to click on the box to find the way to get to goal post.

  • 3. If user press on the wrong box it would be game over then user can play again or they have a chance to go back to the previous state.

  • 4. After finding seven ball user need to click on the goal post to win the game.
